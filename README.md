# Projeto de POO

Este é o projeto final de Programação Orientada a Objetos, desenvolvido em Java.

## Estrutura
- `Autenticação/` - classes de login e autenticação
- `exceptions/` - exceções personalizadas
- `Extras/` - funcionalidades extras como estatísticas, notificações e interface gráfica
- `Projetos/` - classes relacionadas a projetos, busca e relatórios
- `usuario/` - classes de usuário como `Aluno`, `Professor` e `Coordenador`

## Como executar
1. Abra o projeto na sua IDE Java (Eclipse, IntelliJ, VS Code)
2. Compile as classes Java
3. Execute a classe principal do sistema (dependendo de como seu projeto está organizado)
